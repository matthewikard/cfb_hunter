# CFB Hunter

A simple Streamlit-based game for college football fans. The goal is to get the lowest score possible across 8 categories. You are presented with a random college football team and attempt to assign the best category for that team aiming to ultimately score as low as you can.

Inspired by [Geo Hunter](https://www.geohuntergame.com).

The application is deployed at https://cfbhunter.streamlit.app

To run locally:

## Requirements
- Python 3.9+
- `streamlit`

## Running the Game Locally
```
pip install streamlit
streamlit run app.py
```
