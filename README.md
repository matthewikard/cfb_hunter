# CFB Hunter

A simple Streamlit-based game for college football fans.

Inspired by [Geo Hunter](https://www.geohuntergame.com).

The application is deployed at https://cfbhunter.streamlit.app

To run locally:

## Requirements
- Python 3.9+
- `streamlit`

## Running the Game Locally
```
pip install streamlit
streamlit run app.py
```
